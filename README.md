# JsonReader

JsonReader — это Qt-приложение для импорта, парсинга и отображения данных из XML и JSON файлов.  
Программа поддерживает многопоточную обработку, хранение данных в SQLite и удобный интерфейс для редактирования.

## Установка

```bash
git clone https://github.com/WunderwaffleX/JsonReader.git

cd JsonReader
make build

./build/JsonReader

```
